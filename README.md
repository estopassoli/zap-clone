## About


Este projeto será uma rede social de Chat online semelhante ao WhatsApp Web!

## Tecnologias

- HTML
- CSS
- Javascript
- NodeJs

## Início

- Iniciado em 17/04/2022 até Sem data de término.

## Features

- Envio de mensagens de texto
- Envio de emojis
- Envio de audios
- Envio de gravação de voz
- Envio de imagens
- Envio de vídeos
- Envio de GIFs
- Envio de Documentos
